# qr-code-generator checklist

Last updated: 2025-12-06 17:29 MST by Codex

Tags: #skill/qr-code-generator

- [ ] Dependencies installed (`pip install qrcode[pil]`).
- [ ] Input text/URL provided and correct.
- [ ] Output path is writable and uses `.png`.
- [ ] Command runs without errors.
- [ ] QR code scans successfully with a phone or scanner.
- [ ] Activity log updated for meaningful changes.

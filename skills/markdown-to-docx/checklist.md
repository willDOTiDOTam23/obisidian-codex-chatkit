# markdown-to-docx checklist

Last updated: 2025-12-06 17:29 MST by Codex

Tags: #skill/markdown-to-docx

- [ ] Pandoc installed and reachable via `pandoc --version`.
- [ ] Input Markdown path exists and is readable.
- [ ] Output path is writable and uses `.docx` extension.
- [ ] Conversion command succeeds without errors.
- [ ] Generated docx opens and key formatting elements (headings, lists, code) render correctly.
- [ ] Activity log updated for meaningful changes.

# hashtags-guide

Last updated: 2025-12-06 17:29 MST by Codex

## Purpose
- Document baseline tag conventions for this vault.

## Baseline tags
- `#skill/<name>` — skills and their artifacts.
- `#intake/new` — rough notes awaiting normalization.
- `#log/<author>` — attribution for edits (e.g., `#log/codex`).
- `#status/{in-progress,blocked,done}` — optional work state for tasks/notes.
- `#area/<topic>` — topical grouping (e.g., `#area/automation`).

## Usage
- Apply `#log/<author>` to notes you materially change; prefer one per edit session.
- Combine `#status/*` with intake or task notes to reflect current state.
- Update this guide when introducing new tag families; log changes below.
